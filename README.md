# 2048游戏

## 简介

这是一个基于Pygame开发的2048小游戏，可以在电脑上进行游戏。

游戏规则是将相同数字的方块合并，达到更高分数。



## 安装与运行

### 安装

1. 安装Python 3.x版本。
2. 在命令行中运行以下命令，安装需要的依赖库：

```bash
pip install -r requirements.txt
```

### 运行

在项目根目录下执行以下命令启动游戏：

```bash
python game.py
```



## 游戏操作

- 上下左右箭头：控制方块的移动。
- F1键：提前结束游戏并记录最高分。



## 代码结构

- `config.py`：配置文件。
- `game.py`：游戏逻辑实现。
